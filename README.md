#  CNN Chess Piece Classifier

Projeto de Inteligência Artificial para classificação de peças de xadrez usando CNN + API Node.js + Frontend simples.

---

##  Estrutura do projeto

- `server.js` → API Node.js
- `models_saved/model.pth` → modelo treinado
- `frontend/index.html` → interface web simples

---

##  Como executar o projeto

### 1. Instalar dependências
```bash
npm install

npm start

-> abrir o index.html no navegador, selecionar um arquivo de foto que deseja testar e enviar.
